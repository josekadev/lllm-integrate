# Proyecto LLM Lab

Proyecto de pruebas con Docker, frontend web y modelo LLM local.

## Tecnologías

- Docker
- Dev Containers
- Next.js / Web frontend
- Ollama / LLM local

## Requisitos

- Docker
- Docker Compose
- VS Code (opcional)
- Dev Containers (opcional)

## Levantar el proyecto

```bash
docker compose up --build
```
